# 🔐 Secure Task & Productivity Management System

## 📌 Introduction

This project is a secure task management system that allows users to register, log in, and manage their daily tasks efficiently. It ensures data security using password hashing and stores user data locally using SQLite.

---

## 🚀 Features

* 🔐 User authentication (Login & Registration)
* 📝 Add, update, and delete tasks (CRUD operations)
* ✅ Mark tasks as completed
* 📊 Track task status (Pending / Completed)
* 💾 Local database storage using SQLite
* 🔒 Secure password storage using hashing

---

## 🛠️ Tech Stack

* **Python**
* **Streamlit** – UI framework
* **SQLite** – Database
* **Hashlib** – Password encryption

---

## ⚙️ How It Works

1. Users register with a username and password
2. Password is securely hashed and stored
3. After login, users can:

   * Add tasks
   * Mark tasks as completed
   * Delete tasks
4. All data is stored locally in SQLite database

## ⚙️ Installation

### 1. Clone repository

```

```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run application

```
python -m streamlit run app.py
```
